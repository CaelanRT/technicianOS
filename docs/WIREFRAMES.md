# TechnicianOS Wireframe Specifications

Paper MCP can be used to create high-fidelity designs. Open a Paper document and use the plan's design system to build:

1. **Login** – Email/password form, link to register
2. **Register** – Name, email, password, role, org selector
3. **Dashboard/Projects List** – Nav, project cards, create button
4. **Project Detail** – Project info, task list, add task
5. **Task Form** – Name, description, project, status, assignee
6. **Profile** – User edit form

Layout: Desktop 1440px, tablet 768px, mobile 375px. Use design tokens from `src/design-system`.
